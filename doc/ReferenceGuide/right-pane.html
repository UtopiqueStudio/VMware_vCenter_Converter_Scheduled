VMware vCenter Converter Standalone API Reference Guide

Welcome to the API Reference Guide for VMware vCenter Converter Standalone SDK. This document provides comprehensive information about all data structures available through the Converter Standalone API:

  • Managed object types
  • Data object types
  • Properties
  • Enumerations
  • Faults

The Converter Standalone API is exposed as a Web service running on Converter Standalone server systems.

The Converter Standalone API Reference is a core component of the Converter Standalone SDK. The Converter Standalone SDK includes Java and C# samples that demonstrate features of Converter Standalone server. The Converter Standalone SDK enables developers to evolve their applications to support new server features while maintaining compatibility with older server versions.
