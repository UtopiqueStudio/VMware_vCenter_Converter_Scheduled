ConverterServerProtectionJobRebootAndResume

Reboot the protected source and resume protection operations. This should be called only if the source's protection agent has been changed in such a way that a reboot of the machine is required to continue protection operations.

Note: If a time for reboot is specified then a VC scheduled task will be created to trigger this operation at the appropriate time. This task will be set in return returned task's info property as vcTask

Required Privileges
None

Parameters

NameTypeDescription
_thisManagedObjectReference A reference to the ConverterServerProtectionProtectionJob used to make the method call.
timeToRun*xsd:dateTime

An optional time to run this operation. If set then a VC task will be created to initiate the operation. If not set the the operation will be run immediately.
*Need not be set

Return Value

Type Description
ConverterTaskInfoA task instance to use to monitor this reboot and resume operation.

Faults

Type Description
