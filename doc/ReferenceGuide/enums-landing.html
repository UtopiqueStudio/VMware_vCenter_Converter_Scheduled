All Enumerations Overview

Enumerations are predefined values used to store information in enumerated types.

All enumerations in the VMware vCenter Converter Standalone API are listed in the pane to the left of this page. You can also enter part of an enumeration in the Quick Index to find an entry. Entries in this section provide information about which enumerated type includes each enumeration. As a result, there might be multiple entries, with each one applicable to a different enumerated type.
