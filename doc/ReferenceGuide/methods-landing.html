All Methods Overview

The All Methods pane to the left of this page includes information about all methods available in the vCenter Converter Standalone API. Use these methods to complete tasks such as setting or reading values stored as properties of managed objects or data objects.

Methods might include:

  • Description of purpose and usage.
  • Privileges this method requires.
  • Parameters this method accepts.
  • Values this method returns.
  • Faults this method throws.
  • Object types that support this method.

You can also enter part of a method name in the Quick Index to find an entry.
