Enumerated Types Overview

Enumerated types that can store one of a set of predefined values. Enumerated types are a way to specify a set of values that can be passed to or returned from a function.

Enumerated types might include:

  • Objects of which this object is a property.
  • Operations that take this type as a paramter.
  • Description of purpose and usage.
  • A list of valid enum constant values.

All enumerated types are listed in the pane to the left of this page. Click a name to see more information. You can also enter part of an enumerated type name in the Quick Index to find an entry.
