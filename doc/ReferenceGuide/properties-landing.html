All Properties Overview

Properties store information about VMware vCenter Converter Standalone objects.

All properties in the Converter Standalone SDK are listed in the pane to the left of this page. You can also enter part of a property name in the Quick Index to find an entry. Entries in this section include information about which classes include each property. As a result, there might be multiple entries, with each one applicable to a different object type.
