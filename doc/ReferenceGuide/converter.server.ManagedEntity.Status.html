Enum - ConverterManagedEntityStatus

Property of
ConverterManagedEntity
Since
Converter API 2.0


Enum Description

Identifies the entity's status. For a protection entity this is the protection status. Status values are ranked from lowest to highest as: green, yellow, red, gray.

Enum Constants

NameDescription
gray The entity's status is unknown.
green The entity is in a healthy state.
red The entity is currently in an error state and requires attention.
yellow There may be a problem with the entity. For example, for a protection entity this can be triggered due to missing RPO, etc. The user may wish to analyze the problem and perform corrective actions on their own.
