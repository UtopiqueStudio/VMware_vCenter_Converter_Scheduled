secondaryProtectionConverterSRMVmProtectionInfo

The current status of secondary replication for this job's target VM. The result structure includes: * srm.VmProtectionInfo.isConfigurable : boolean : Whether or not the target virtual machine can be replicated via SRM. * srm.replicationMechanism : srm.ReplicationMechanism : The current secondary replication mechanism for the target virtual machine. Can be none, sanReplication, or hbrReplication (HBR is not supported in version 1). * srm.VmProtectionInfo.protectionGroupName : String : The name of the secondary replication's protection group protecting the target virtual machine. Only valid once secondary replication has been configured
